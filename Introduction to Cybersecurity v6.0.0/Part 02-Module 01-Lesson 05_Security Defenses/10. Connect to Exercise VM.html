Connect to Exercise VM

Student lab set up

You will get remote control of a virtual Windows 10 PC called CYBERND01PRJ01 and complete a few exercises. The setup may take 20 minutes. Please be patient!

The following are the materials you need to set up the virtual PC

  1. Initialize-Udacity.CYBERND01.LabVM2.ps1 to install the virtual PC called "CYBERND01PRJ01". Click here to download the file and unzip it when the download is completed.
  2. Username and password to log in CYBERND01PRJ1
  • Username: cyberadmin
  • Password: @UdacityLearning#1
  1. Set up the project virtual PC "CYBERND01PRJ01". Follow the instruction below to set it up. You will need Initialize-Udacity.CYBERND01.LabVM2.ps1, the username and password provided to complete the setup.

Before installation, please make sure you have met all the prerequisites listed below. If you have met the prerequisites, please follow the instruction: Setup Student LAB to set it up and get the remote control.

Note: the setup process may take about 30 min to finish. Please be patient and have fun.

IMPORTANT: Please always remember to shut down ALL of the virtual machines when not in use to avoid charges! The status of ALL virtual machines should be stopped!
