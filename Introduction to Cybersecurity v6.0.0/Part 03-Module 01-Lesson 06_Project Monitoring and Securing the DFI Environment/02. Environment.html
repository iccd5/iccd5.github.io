Setting Up Your Project Environment

An Azure environment has been created for this project. You will need to step an Azure VM and use the image provided in the instructions below to complete this project.

Use the Lesson resources (in the top left of the classroom) to download the files you will need to complete this project. Please use the task list below to help you with setting up your VMs.
